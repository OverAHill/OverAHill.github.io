Academic Projects

Projects made for university according to a set specification


Tank AI Project

For my AI module I created a tank AI with the aim of competing in a group arena battle; I programmed many steering behaviours which were controlled using a fuzzy state machine and pathfinding done via the Dynamic Recursion Search algorithm. Achieved: 100%


Colour Theory

5 Person group project in Unreal, a physics based puzzle game inspired by portal, the player changes the properties of cubes using colour and must solve a variety of puzzles. Achieved: 80%


Physics Engine

My physics engine built in DirectX 11, contains a mass aggregate particle system, rigid bodies, a variety of forces and collision handling. Achieved: 93%


Scott Vs Networking

My First ever Project in C#! A networked chat client using windows forms that uses techniques such as multithreading and thread locks, alongside a networked game that allows two users to play a more basic version of the fan favourite Scott Pilgrim vs the World the game online. Achieved: 86%!


Hyrule Hack n' Slash

Unreal 4 project made as part of a pair, for my part I created a camera controller, combo attack system, mini map, weapon damage, all UIs, and a non-possessed player controller!. Achieved: 100%.


Rocket Ball

My DirectX 11 Project, the graphical half; makes use of phong shading, specular mapping, and billboarding. Achieved: 89%.


Apple Game in OpenGL

In C++ I focused on creating a first person player movement system that uses matrices to create smooth movement. The game has a basic particle effect for smoke on the fire that activates based on user input. Module Mark: 72%


Retro Mario: SDL2 Project

For Games Engine Project two we were tasked with producing a clone of mario using my own engine, built uisng SDL2, I replicated the arcade version with two players collecting coins to beat a high score. Then created a solo mode boss battle against a familiar villain. Module mark: 95%


Cat-aclysm: S2D Project

My first game made using C++ and the university's proprietary API, S2D. It is a basic collect game vs multiple AI; one that collects the same object as you and another that aims to take away your lives. Low frame rate animation and menus are also implemented features that I created. Module Mark: 72%
